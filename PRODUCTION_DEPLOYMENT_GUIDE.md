# Daily Tutors - Production Deployment Guide

## Overview
Daily Tutors is a production-ready AI-powered UPSC preparation platform built with Next.js 15, MongoDB, NextAuth.js, Razorpay, and automated AI content generation.

## Prerequisites

### Required Services
1. **MongoDB Atlas** - Cloud database
2. **Vercel Pro** - Hosting & deployment
3. **Razorpay** - Payment processing
4. **Groq Cloud** - AI API for content generation
5. **Google OAuth** - Authentication
6. **GitHub** - Source control & CI/CD

## Step-by-Step Deployment

### 1. Database Setup (MongoDB Atlas)

```bash
# Create MongoDB Atlas Account
# 1. Go to mongodb.com/cloud/atlas
# 2. Create a new cluster
# 3. Create a database user
# 4. Get connection string

# Connection string format:
# mongodb+srv://username:password@cluster.mongodb.net/dailytutors?retryWrites=true&w=majority
```

### 2. Authentication Setup

#### Google OAuth
```bash
# 1. Go to console.cloud.google.com
# 2. Create a new project
# 3. Enable Google+ API
# 4. Create OAuth 2.0 credentials (Web application)
# 5. Add authorized redirect URIs:
#    - http://localhost:3000/api/auth/callback/google (local)
#    - https://yourdomain.vercel.app/api/auth/callback/google (production)
# 6. Copy Client ID and Secret
```

#### NextAuth Secret
```bash
# Generate a secure secret
openssl rand -base64 32
```

### 3. Payment Integration (Razorpay)

```bash
# 1. Sign up at razorpay.com
# 2. Go to Settings > API Keys
# 3. Copy Key ID and Key Secret
# 4. For testing, use test keys
# 5. For production, verify your business details
```

### 4. AI API Setup (Groq)

```bash
# 1. Go to console.groq.com
# 2. Create API key
# 3. Save for environment variables
```

### 5. Environment Variables

Create `.env.local` file:

```env
# Database
MONGODB_URI=mongodb+srv://user:pass@cluster.mongodb.net/dailytutors

# NextAuth
NEXTAUTH_SECRET=your-generated-secret-here
NEXTAUTH_URL=https://yourdomain.vercel.app

# Google OAuth
GOOGLE_CLIENT_ID=your-client-id
GOOGLE_CLIENT_SECRET=your-client-secret

# Razorpay
NEXT_PUBLIC_RAZORPAY_KEY_ID=rzp_live_xxxxx
RAZORPAY_KEY_SECRET=your-secret-key

# AI APIs
GROQ_API_KEY=your-groq-key

# App Settings
NEXT_PUBLIC_APP_URL=https://yourdomain.vercel.app
CRON_SECRET=your-random-cron-secret
```

### 6. Local Development

```bash
# Install dependencies
npm install

# Set up environment variables
cp .env.local.example .env.local
# Edit .env.local with your secrets

# Run development server
npm run dev

# Open http://localhost:3000
```

### 7. Deploy to Vercel

#### Option A: Using Vercel Dashboard
```bash
# 1. Go to vercel.com
# 2. Click "New Project"
# 3. Import GitHub repository
# 4. Set environment variables
# 5. Click "Deploy"
```

#### Option B: Using Vercel CLI
```bash
# Install Vercel CLI
npm i -g vercel

# Login to Vercel
vercel login

# Deploy
vercel --prod

# Set environment variables in dashboard
```

### 8. Configure GitHub Actions

```bash
# Add secrets to GitHub repository
# Settings > Secrets and variables > Actions

# Add these secrets:
- VERCEL_TOKEN
- VERCEL_ORG_ID
- VERCEL_PROJECT_ID
- MONGODB_URI
- NEXTAUTH_SECRET
- NEXTAUTH_URL
- GOOGLE_CLIENT_ID
- GOOGLE_CLIENT_SECRET
- NEXT_PUBLIC_RAZORPAY_KEY_ID
- RAZORPAY_KEY_SECRET
- GROQ_API_KEY
- CRON_SECRET
```

### 9. Configure Vercel Cron Jobs

Cron jobs are configured in `vercel.json`:

```json
{
  "crons": [
    {
      "path": "/api/cron/fetch-news",
      "schedule": "0 */1 * * *"  // Every hour
    }
  ]
}
```

Add `CRON_SECRET` to Vercel environment variables for security.

## Database Models

The platform includes these MongoDB collections:

- **users** - Student and admin accounts
- **courses** - Course information
- **enrollments** - Student course enrollments
- **articles** - Current affairs articles (auto-generated by AI)
- **quizzes** - Quiz questions and tests
- **quizAttempts** - Quiz attempt records
- **payments** - Payment transactions
- **subscriptions** - Subscription plans
- **magazines** - Monthly PDF magazines
- **notes** - Student notes
- **categories** - Article categories

## API Routes

### Authentication
- `POST /api/auth/register` - User registration
- `POST /api/auth/login` - User login
- `POST /api/auth/logout` - User logout

### Courses
- `GET /api/courses` - List courses
- `POST /api/courses` - Create course (admin)
- `GET /api/courses/[id]` - Get course details
- `PUT /api/courses/[id]` - Update course (admin)

### Articles
- `GET /api/articles` - List articles
- `POST /api/articles` - Create article (admin)
- `GET /api/articles/[slug]` - Get article details

### Quizzes
- `GET /api/quizzes` - List quizzes
- `POST /api/quizzes` - Create quiz (admin)
- `POST /api/quizzes/[id]/submit` - Submit quiz attempt

### Payments
- `POST /api/payments/create-order` - Create Razorpay order
- `POST /api/payments/verify` - Verify payment

### Automation
- `GET /api/cron/fetch-news` - Fetch and process news (runs every hour)
